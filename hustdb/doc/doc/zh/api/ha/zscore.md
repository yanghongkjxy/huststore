## zscore ##

**接口:** `/zscore`

**方法:** `GET | POST`

**参数:** 

*  **tb** （必选）  
*  **key** （必选）  

该接口是 `/hustdb/zscore` 的代理接口，参数详情可参考 [这里](../hustdb/hustdb/zscore.md) 。

**使用范例:**

    curl -i -X POST "http://localhost:8082/zscore?tb=test_table" -d "test_key"
